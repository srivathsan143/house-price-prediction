# House Price Prediction

## Overview
This project predicts house prices using Machine Learning.

## Features Used
- Area (square feet)
- Number of bedrooms

## Model
Linear Regression

## Workflow
1. Load dataset  
2. Train model  
3. Save model  
4. Predict price  

## How to Run
pip install -r requirements.txt  
python train.py  
python predict.py  

## Sample Output
Predicted Price: 440000 (example)
